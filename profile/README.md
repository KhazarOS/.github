# Welcome to KhazarOS Ecosystem 🌊

Təşkilatımızın əsas məqsədi istifadəçilər üçün gündəlik həyatda lazım olan kiçik, sürətli və effektiv tətbiqlər (micro-apps) hazırlamaqdır. Bu tətbiqlər gələcəkdə **KhazarOS** əməliyyat sisteminin ayrılmaz hissəsi olacaq, lakin hazırda digər bütün əməliyyat sistemlərində də tam funksional şəkildə istifadə edilə bilər.

## 🚀 Məqsədimiz
Biz sadəliyi və funksionallığı ön planda tuturuq. Hazırladığımız hər bir alət həm fərdi istifadə üçün, həm də KhazarOS daxilində modulyar komponent kimi nəzərdə tutulub.
